## 2. 宠物档案设计

8 只宠物覆盖不同场景：
- **Mochi** (布偶猫, 母) — 粘人爱睡觉，需要特别照护
- **土豆** (英短, 公) — 活泼贪吃，偷吃蛋糕的调皮猫
- **Lucky** (金毛, 公) — 心脏偏大需要长期服药，训练进度最快
- **奶茶** (泰迪, 母) — 有皮肤过敏，定期美容
- **橘子** (田园猫, 公) — 高冷独立，已绝育
- **豆豆** (柴犬, 公) — 固执表情丰富，散步爱罢工
- **雪花** (俄蓝, 母) — 认生优雅，怕陌生人
- **巧克力** (拉布拉多, 母) — 5 岁老狗，关节炎需要保健品

## 3. 异常状态设计

刻意设计了以下需要关注的状态：
- **库存不足**：皇家猫粮 0.8kg、pidan 猫砂仅剩 1 袋、尼可信仅 1 粒、除臭剂快用完
- **疫苗过期**：Lucky 犬四联和狂犬疫苗已过期，土豆猫三联已过期
- **驱虫过期**：豆豆体外驱虫已过期
- **健康异常**：橘子食欲下降、巧克力关节问题、Lucky 运动后喘粗气

## 4. 数据真实性保障

- 所有品种使用真实常见品种
- 疫苗/驱虫间隔符合兽医建议
- 药品名称使用真实通用名（匹莫苯丹、软骨素等）
- 品牌使用真实宠物品牌（渴望、皇家、Kong 等）
- 开销金额参考真实市场价格
- 日记内容真实自然，贴近养宠日常
